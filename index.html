Sobre Mi

Mi nombre es Betsabé De Aquino Pascual y tengo 16 años. Soy estudiante de Nivel Medio Superior, estudio la especialidad de Programación, actualmente me encuentro en 4to Semestre. Mis estudios los estoy llevando acabo en el Plantel Cetis 43 "Mariano Matamoros Guridi"

  • Enero 2018 - Junio 2018

    Programación Estructurada

    Fue un curso de 6 meses, en el que Aprendí las Bases de la Programación, que es programación y para que sirve, las estructuras Selectivas, los Ciclos, la Estructura de un Programa. Interactue con entornos como P-seint y Dev-C++. Aprendi el Lenguaje C y algo de C++.

  • Agosto 2018 - Diciembre 2018

    POO Y Java

    En este curso aprendí un Nuevo Lenguaje de Programación "JAVA" que es un Lenguaje Orientado a Objetos, Estuve trabajando con animaciones y con Ventanas denominadas JFrame, en las cuales diseñaba una ventana con ciertos objetos (esto da mas estetica al Programa), con Java programe estos objetos, para que funcionacen. Tambien enlazamos estos pequeños programas con una Base de Datos. Los entornos con los que interactue fueron Alice, NetBeans, Xampp y phpMyAdmin.

  • Enero 2019 - Junio 2019

    Desarrollo de Sitios Web

    Me encuentro en medio de este curso en el que estoy aprendiendo un Lenguaje de Etiquetas HTML Y HTML5; asi como CSS que se usa para dar diseño a la pagina. La impartidora del curso la Profra. Norma tambien nos enseñara Java Script asi como PHP que también son importantes para el Desarrolo de un Sitio Web mas Bonito y Funcional. Los entornos que hemos y estamos utilizando son Bloc de Notas, Sublime Text, Xampp, File Zilla, FTP y Bootstrap.

  • Enero 2019 - Junio 2019

    Desarrollo de A. Moviles

    Me encuentro aprendiendo a diseñar aplicaciones moviles, para ello estamos retomando el Lenguaje de Programación "Java", asi mismo estamos ocupando un Archivo denominado XML en el que se guarda nuestro diseño gráfico. Los entornos con los que he interactuado son Android Studio y App Inventor.
